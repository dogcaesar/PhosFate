PhosFate

An Atlas of Human Conditional Phosphoregulation

Profiling Conditional Kinase Activities

{% image activities.png %}

To describe the kinase regulation in a panel of 150 conditional phosphoproteomes, we systematically predicted the conditional activity of each kinase by testing the enrichment of previously reported substrates on differentially regulated phosphosites. This statistical analysis is based on the intuition that a kinase activity in a given condition is increasing if its known target sites tend to be up-regulated. Alternatively to previous efforts, we used a modified version of the Kinase Set Enrichment Analysis (KSEA) which avoids arbitrary thresholds and resembles the original Gene Set Enrichment Analysis (GSEA). As a result, we predicted the activity for a list of 209 kinases in a range between 10 to 150 conditions.

A Map of Human Signaling States

The kinase activity profiling results useful to discriminate similar responses based on their activity changes after perturbation. Motivated by these idea, we developed a robust algorithm to order the high-dimensional kinase activities into an interpretable map that could cluster biological states and reveal progression through different signaling processes. We then use the signaling map to search for groups of conditions displaying similar kinase activity profiles as well as to find kinases specifically regulated in certain molecular responses.

Effector Complexes Regulation

{% image complex.png %}

In order to associate drivers and effectors, we looked for protein complexes presenting a synchronized regulation with kinases across conditions. In order to obtain these predictions we first calculated the undergoing complex regulation by quantifying the enrichment on highly phospho-regulated sites using the Kolmogorov-Smirnov test (KS). We observed that kinases under active regulation in the same conditions as the protein complexes could directly or indirectly drive their molecular response.
